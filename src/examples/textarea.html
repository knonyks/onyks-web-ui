Onyks Textarea

A multi-line text input for general purposes, e.g., for forms or comments.

1. Default textarea

2. Pre-filled value

3. Defined rows and cols

4. Character constraints (minlength / maxlength)

5. Disabled state

Prevents user interaction completely.

6. Read-only state

User can read and copy the text, but cannot modify it.

7. Required state

8. Fixed size (No resize)

User cannot drag the corner to resize the textarea.

9. Error state

Visual indication of invalid input.

10. Rozmiary (Sizes)

Kontrola wielkości tekstu i wewnętrznych odstępów za pomocą atrybutu size="s|m|l".

Rozmiar S (Small)

Rozmiar M (Medium / Default)
